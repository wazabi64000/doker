Tutoriel Docker pas-à-pas

Dans ce tutoriel, nous allons créer et lancer notre premier conteneur Docker, étape par étape, avec tests pour chaque étape.

Étape 1 : Vérifier l'installation de Docker


docker --version
docker run hello-world

Si vous voyez un message de bienvenue de Docker, votre installation est correcte.

Étape 2 : Télécharger une image Docker


docker pull nginx
docker images

Vous devez voir l'image nginx dans la liste.

Étape 3 : Lancer un conteneur Nginx


docker run -d -p 8080:80 --name mon-nginx nginx

Ouvrez votre navigateur et allez sur http://localhost:8080. Vous devriez voir la page d'accueil Nginx.

Étape 4 : Lister les conteneurs en cours


docker ps

Vous devez voir mon-nginx en cours d’exécution.

Étape 5 : Arrêter et supprimer le conteneur


docker stop mon-nginx
docker rm mon-nginx

Le conteneur est maintenant arrêté et supprimé.

Étape 6 : Créer votre premier Dockerfile


# Créer un fichier Dockerfile
nano Dockerfile

# Exemple de contenu :
FROM nginx
COPY index.html /usr/share/nginx/html/

Construisez ensuite votre image personnalisée :


docker build -t mon-nginx-personnalise .
docker run -d -p 8080:80 mon-nginx-personnalise

Vérifiez que votre page personnalisée s’affiche sur http://localhost:8080.
