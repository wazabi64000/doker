Installation de Docker

Suivez les instructions correspondant à votre système d’exploitation pour installer Docker et tester qu’il fonctionne correctement.

1. Linux (Ubuntu/Debian)


# Mettre à jour le système
sudo apt update
sudo apt install apt-transport-https ca-certificates curl software-properties-common

# Ajouter la clé Docker et le dépôt
curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg
echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

# Installer Docker
sudo apt update
sudo apt install docker-ce

# Vérifier l'installation
docker --version
docker run hello-world

2. Windows

  1. Téléchargez Docker Desktop.
  2. Installez-le en suivant les instructions.
  3. Ouvrez PowerShell et vérifiez :

docker --version
docker run hello-world
