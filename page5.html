Exercice pratique : Créer votre site web Dockerisé

Dans cet exercice, vous allez créer un mini-site web HTML et le déployer dans un conteneur Docker Nginx.

Étape 1 : Créer les fichiers du projet


mkdir mon-site
cd mon-site
nano index.html

Exemple de contenu pour index.html :





  
  Mon site Docker


  
Bienvenue sur mon site Dockerisé !

  
Exercice pratique terminé avec succès.

Étape 2 : Créer un Dockerfile


nano Dockerfile

Exemple de contenu :


FROM nginx
COPY index.html /usr/share/nginx/html/

Étape 3 : Construire et lancer le conteneur


docker build -t mon-site-docker .
docker run -d -p 8080:80 mon-site-docker

Ouvrez votre navigateur et allez sur http://localhost:8080 pour voir votre site.

Étape 4 : Vérifier et nettoyer


docker ps          # Vérifier le conteneur en cours
docker stop     # Arrêter le conteneur
docker rm       # Supprimer le conteneur
docker rmi mon-site-docker  # Supprimer l'image

Challenge bonus

Essayez de créer un **volume Docker** pour stocker un fichier texte modifiable, afin que les modifications persistent même si le conteneur est supprimé.
